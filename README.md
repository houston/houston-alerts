# Houston::Alerts

A module for Houston to show and facilitate alerts


## Installation

In your `Gemfile`, add:

    gem "houston-alerts"

And in `config/main.rb`, add:

```ruby
use :alerts do
  # TODO: specify configuration options for Houston::Alerts here
end
```

And then execute:

    $ bundle
